CSS Properties used

Background Properties

  • background-color: color name/hex value/rgb value
  • background-image: url("path to image")
  • background-repeat: no-repeat

Box Model

  • width: specify width in pixels or percentage
  • breadth: specify breadth in pixels or percentage
  • border: specify style(dotted, solid etc)| width| color
  • padding: specify the gap between the content and the html element border
  • margin: specify the gap between different html elements, as margin-top/-bottom/-right/-left

Font

  • color: color name/hex value/rgb value
  • font-size: specify size in pixels or
  • font-family: specify multiple font names to select the first one available from the list
  • font-style: italic/ normal/ oblique

Text

  • text-align: specify alignment of text in the lement, like right/left/center
  • text-transform: specify case sensitivity of text, like uppercase/ lowercase
  • letter-spacing: specify spacing in pixels
  • line-height: specify line spacing in pixels

Float

  • float: left/right
  • float is different from align as align will justify the content in the element, whereas float justifies the element itself
